A Simple Grid

When ever you add the CSS property display:grid to a container, all it's children will automatically transform into grid units.

Note: Nothing changes until you tell the grid how many rows and columns you need and that's exactly what we will do in the next step.

Grid Cell.

A grid cell is an area created when columns and rows intersect inside a grid container.

CSS Snippets:

  • Columns: grid-template-columns
  • Rows : grid-template-rows 
        .two {
            grid-template-columns: 100px 100px 100px;
        }

Here, we are telling the grid container to add 3 columns and 3 rows of 100px each.

Note: The grid container will automatically create the Rows if it is not defined and the height is set to fit the content inside it.

Fractional unit

If there is any space left inside the grid container after displaying the grid cells (as per the values given by the user) we can refer to this space as Fracional unit. Just like a "px" or "%" we can use "fr" to display the grid unit in the free space. 

 
        .three {
            grid-template-columns: 1fr 1fr 3fr;
        }

In the below example we have set the 2nd column to "1fr" and 3rd col to 3fr. Which means the 3rd col will take 3 times the available space compared to the 1st col.

The CSS grid system will automatically place the rest of the child elements below the available grid cells to complete the grid. The child elements inherits the column width in which it is automatically placed.

Grid Span

By default, the elements are layed out from left to right inside the grid.

Spanning can address the issue we faced during the fractional unit example where all the rows in a column has the same width. But we can change this using the span property to give each grid unit a different width.

We can also tell the grid to start and end a cell at a specific points.

In the below example we span the 5th column to take up three times as much the width and twice the height of the rest of the units. 

        .item5 {
            grid-column-start: 2;
            grid-column-end: 5;
            grid-row-start:2;
            grid-row-end:4;
        }
And the shorthand CSS for the above is: 
        .item5 {
            grid-column: 2/5;
            grid-row:2/4;
        }

Sometimes you may not be aware of how many rows or columns exist and ou want to span a grid item across an entire width. You can do it by using -1 as the end point.

CSS to span accross entire width. -1 does not work to span the entire height. 

        .item5 {
            grid-column: 2/-1;
            grid-row:2/-1;
        }
